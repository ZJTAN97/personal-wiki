## Approach

### What is the Two Pointers Technique?

The **two pointers** technique uses two pointers (indices) that traverse an array in a coordinated way. In this problem:
- **Right pointer**: Always moves forward with each iteration (scans for non-zero elements)
- **Left pointer**: Only moves forward when we place a non-zero element

This is essentially a variant of the "partition" step in quicksort - separating the array into two regions.

### Why This Works

Instead of creating a new array, we use the two-pointer technique to:
1. Collect all non-zero elements at the front (left side)
2. Fill the remaining positions with zeros (right side)

### Algorithm Steps

1. **Left pointer starts at 0**: This marks where the next non-zero element should go
2. **Right pointer scans all elements**: Looking for non-zero values
3. **When non-zero found**: Swap with left pointer position, then advance left pointer
4. **Continue**: Until right pointer finishes scanning

### Key Concept - It's a Swap, Not Pointer Swap

```
❌ WRONG: Swapping the pointers themselves
   leftPointer ↔ rightPointer

✓ CORRECT: Swapping the VALUES at those positions
   nums[leftPointer] ↔ nums[rightPointer]
   
Note: We only swap when we find a non-zero element!
```

### Key Pattern for This Problem

```typescript
function moveZeroes(nums: number[]): void {
  let leftPointer = 0;

  for (let rightPointer = 0; rightPointer < nums.length; rightPointer++) {
    // Found a non-zero element
    if (nums[rightPointer] !== 0) {
      // Swap with left pointer position
      [nums[leftPointer], nums[rightPointer]] = [nums[rightPointer], nums[leftPointer]];
      
      // Advance left pointer to next position
      leftPointer++;
    }
  }
}
```

### When to Use This Pattern

- Moving elements to separate regions (non-zeros vs zeros)
- Partitioning arrays based on a condition
- Removing certain elements while maintaining relative order

## Solution

```ts
function moveZeroes(nums: number[]): void {
  let leftPointer = 0;
  let temp: number;

  for (let rightPointer = 0; rightPointer < nums.length; rightPointer++) {
    if (nums[rightPointer] !== 0) {
      temp = nums[leftPointer];
      nums[leftPointer] = nums[rightPointer];
      nums[rightPointer] = temp;

      leftPointer++;
    }
  }
}
```
