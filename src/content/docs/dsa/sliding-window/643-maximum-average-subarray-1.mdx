## Approach

### What is a Fixed Sliding Window?

A **fixed sliding window** is used when you need to process consecutive elements of a fixed size k. Instead of recomputing the sum for each window from scratch (O(nГ—k)), we slide the window and update the sum in O(1) time.

### Why This Problem Uses Sliding Window

We need to find the maximum average of any contiguous subarray of length k. The brute force approach would calculate the sum of each k-length subarray separately, resulting in O(nГ—k) time. With sliding window, we achieve O(n) time.

### Algorithm Steps

1. **Initialize**: Calculate sum of first k elements (first window)
2. **Slide**: Move window one position right, update sum by removing left element and adding new right element
3. **Track**: Keep track of maximum average seen

### Key Pattern for Sliding Window

```typescript
// Initialize first window
let windowSum = nums.slice(0, k).reduce((a, b) => a + b, 0);
let maxAvg = windowSum / k;

// Slide the window
for (let i = k; i < nums.length; i++) {
    // Add new element (right side)
    windowSum += nums[i];
    // Remove old element (left side)
    windowSum -= nums[i - k];
    // Update max
    maxAvg = Math.max(maxAvg, windowSum / k);
}
```

### When to Use Fixed Sliding Window

- Need to process consecutive elements of fixed length
- Want O(n) instead of O(nГ—k)
- Looking for max/min/sum of each window


## Solution

```ts
function findMaxAverage(nums: number[], k: number): number {
  if (nums.length === 1) return nums[0];

  let window = nums.slice(0, k).reduce((a, b) => a + b, 0);
  let maxAverage = window / k;

  for (let i = k; i < nums.length; i++) {
    window = window + nums[i] - nums[i - k];
    maxAverage = Math.max(maxAverage, window / k);
  }

  return maxAverage;
}

findMaxAverage([1, 12, -5, -6, 50, 3], 4); // 12.7500
findMaxAverage([0, 1, 1, 3, 3], 4); // 2.0000
```
