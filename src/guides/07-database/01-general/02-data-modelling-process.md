# Data Modelling Process

# Introduction

Data Modelling is the process of analyzing data and creating a desing. There are three types of data models

1. Conceptual
2. Logical
3. Physical

# Steps

1. Determine the Goal of the Database

2. Consider existing / current system

3. Gather requirements of the Database

4. Finding exceptions to the rules

5. Identifying entities

- Entities are things that we want to store data about
- Good way to identify them is to look for nouns in our goal statement

6. Defining the attributes

- columns or fields
- attributed to "object" or entity
