# Properties

## 1. Text Properties

Control the typography of text elements.

- `color`: Sets the text color.
- `font-family`: Specifies the font type.
- `font-size`: Defines the size of the text.

## 2. Box Model Properties

Manage spacing and layout.

- `margin`: Controls the space outside an element.
- `padding`: Sets the space inside an element, between the content and border.
- `border`: Defines the border around an element.

## 3. Background Properties

Style the background of elements.

- `background-color`: Sets the background color.
- `background-image`: Specifies an image as the background.

## 4. Layout Properties

Influence how elements are positioned and displayed.

- `display`: Controls how an element is displayed (e.g., block, inline).
- `position`: Defines how an element is positioned in relation to its parent or viewport.
